# satertALHATAB - MVP Website 🍔🔥

A responsive, high-performance web application designed as a Minimum Viable Product (MVP) for **satertALHATAB** fast-food restaurant. The platform bridges the gap between digital menus and modern messaging workflows, enabling users to seamlessly browse items and place automated orders.

## ✨ Features
* **Modern Food Menu Interface:** Clean, highly visual, and fully responsive presentation of the restaurant's fast-food menu items.
* **WhatsApp API Integration:** Embedded logic that automatically packages the user's selected food items and routes the order placement directly to the restaurant's WhatsApp business line.
* **Mobile-First UX/UI:** Optimized specifically for seamless navigation on smartphones and web browsers alike.
* **Lightweight Architecture:** Built using pure front-end web standards to guarantee lightning-fast load times.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and content layout.
* **CSS3:** Custom responsive styling and modern layout alignment.
* **JavaScript (Vanilla):** Dynamic DOM manipulation and WhatsApp order API payload construction.
